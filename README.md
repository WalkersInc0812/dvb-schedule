# dvb-schedule

## Using

- [pnpm](https://pnpm.io)
- [Next.js](https://nextjs.org/)
- [shadcn/ui](https://ui.shadcn.com/)

## Running Locally

1. Install dependencies using pnpm:

```sh
pnpm install
```

2. Copy `.env.example` to `.env` and update the variables.

```sh
cp .env.example .env
```

3. Start the development server:

```sh
pnpm dev
```
